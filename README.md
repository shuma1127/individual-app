# 苦労した点

制作していくうちにコンセプトを考えれば「ここはこうしたほうがいい」「この機能を付けたい」と設計とは形が異なり
その都度修正を加えて行かなくてはならなくなりました。
そのため無駄な時間が多くかかってしまったことが苦労しました。
なのでアプリ制作をする際は設計はもっと深掘りしてから作成しようと思いました。
